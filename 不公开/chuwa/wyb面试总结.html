介绍与背景

Can you please introduce yourself?

I’m Ethan. I’m a senior Java developer specializing in the design and deployment of web-based applications and distributed systems. My expertise lies in Java and modern web technologies, such as Spring MVC, Spring Boot, and React. I’ve gotten my hands dirty with both SQL and NoSQL databases, stuff like MySQL, Oracle, MongoDB, and Redis. I’ve also used testing frameworks like JUnit and Mockito. I have hands-on experience with containerization tools, including Docker and Kubernetes. I also have work experience with microservice architecture and its components, including service discovery, API gateways, message queues, and so on.

Your recent project

My latest project is for AT&T ‘s shopping platform. It’s a microservice-based web application. The major services include stock dispatching, direct delivery, store delivery, and item return service, etc. Most of the services are implemented with Spring Boot and Hibernate. By the way, the platform is completely AWS-based. We use AWS RDS for structured data, DynamoDB for document data, Kafka for communication between services, and we deployed and managed the services with ECS and EKS. I mainly focused on upgrading the store delivery service and delivery tracking. This included interacting with our real-time inventory system, which uses Kafka to handle inventory update requests. I also participated in the business logic implementation from the controller to the service layer, and also the repository layer for interaction with databases.
